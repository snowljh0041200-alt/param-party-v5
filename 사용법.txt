월하 · 연가 · 연희 파티모집 v12 PRIVATE FARMING

추가 기능:
- 관리자에서 일반유저/문파원 권한 전환
- 파밍 탭/글/작성은 문파원만 가능
- 파밍 글은 삭제하지 않는 이상 자동삭제되지 않음
- 기본 파밍 아이템:
  해골왕: 해뼈
  어금니: 흑룡 어금니, 묵룡 어금니, 진룡 어금니, 감룡 어금니
- 관리자에서 파밍 아이템 목록 수정 가능
- 득템/노득 기록
- 판매금액, 선입/후입, 후입비율 자동 분배 계산

배포:
git add .
git commit -m "v12 private farming board"
git push

사용:
1. /admin 로그인
2. 회원 관리에서 문파원 버튼 클릭
3. 문파원만 파밍 탭 확인 가능
