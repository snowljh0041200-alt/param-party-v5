v13.7 FARMING SHARE WEIGHT SETTLEMENT

수정 내용:
- 후집합 퍼센트 방식 제거
- 지분 방식으로 변경
- 선집합 지분 기본값: 1.0
- 후집합 지분 기본값: 0.88
- 선/후집합 지분은 작성자가 수정 가능

계산식:
총지분 = 선집합 인원 x 선집합 지분 + 후집합 인원 x 후집합 지분
1지분 금액 = 판매금액 / 총지분
선집합 인당 = 1지분 금액 x 선집합 지분
후집합 인당 = 1지분 금액 x 후집합 지분

예:
판매금액 26,000,000
선집합 5명 / 후집합 1명
선지분 1.0 / 후지분 0.88
=> 선집합 약 4,422,000 / 후집합 약 3,891,000
