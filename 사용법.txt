v40.8 VOICE UNLOCK FIX

기준:
- v40.7

수정:
- 보스 30분/15분/5분 음성 안내 보강
- 페이지 클릭/키입력 시 음성 엔진 자동 활성화
- 크롬 speechSynthesis 멈춤 방지용 cancel/resume 처리
- 한국어 음성 우선 선택
- 음성이 막힌 환경에서는 삑 소리 fallback
- ?voice_test=1 로 음성 테스트 가능

테스트:
1. 배포 후 사이트 접속
2. 페이지 아무 곳이나 한 번 클릭
3. 주소 뒤에 ?voice_test=1 붙여서 접속
   예: https://주소.onrender.com/?voice_test=1
4. "보스 알림 음성 테스트입니다."가 들리면 정상

주의:
- 크롬 정책상 페이지를 한 번 클릭해야 음성이 안정적으로 나올 수 있음.
- PC 자체 볼륨/크롬 사이트 음소거도 확인 필요.
